- Repositioned Neuroplast as an explicit implementation of the Local Cognitive Protocol (LCP) and added normative references to the official LCP repository.

- Generalized planning language away from software-only assumptions and verified the updated package with `npm test` and `npm run validate`.
- Synced the canonical package version statement across README, LCP profile artifacts, and LCP-facing documentation to match package version `1.2.0`.
