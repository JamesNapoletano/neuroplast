# Installed Readmes Work Better When They Start With Immediate Next Steps
#learning

## Insight
An installed workflow README is more useful when its first lines answer "what do I do now?" instead of starting with only descriptive context.

## Reusable Practice
- Lead with a one-sentence orientation that tells the reader they are in the right place and what to do next.
- Separate human and AI startup guidance when both audiences use the same installed files.
- Include a tiny prompt-behavior section when short prompts are part of the workflow so people do not need to infer routing rules from deeper contract files.
