# Path Standardization Learning
#learning

## Insight
When migrating repository conventions (for example, legacy folder names to canonical paths), perform a full-link and text audit before and after edits to avoid broken wiki-links and stale references.

## Reusable Practice
- Run a repository-wide search for legacy terms before changes to scope impact.
- Apply updates consistently across architecture docs, plans, and instruction files.
- Re-run repository-wide search afterward to verify complete removal.
- Ensure bidirectional links between plan and changelog entries are maintained after path migrations.
