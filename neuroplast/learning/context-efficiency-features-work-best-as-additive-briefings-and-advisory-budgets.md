# Context Efficiency Features Work Best As Additive Briefings And Advisory Budgets
#learning

## Insight
Context-efficiency improvements fit Neuroplast best when they compress current-state startup context through additive briefing artifacts and advisory validation budgets instead of changing the canonical contract or adding provider-tuning metadata.

## Reusable Practice
- Add compact briefing artifacts that point back to canonical plans, concept notes, changelog entries, and architecture files.
- Keep bootstrap optimization rules advisory and layered after the mandatory contract-first startup sequence.
- Prefer lightweight validation warnings for context bloat before introducing stricter enforcement.
- Document context-depth behavior in installed guidance and shared adapter bootstrap assets so the optimization remains portable.
