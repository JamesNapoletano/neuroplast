# Human-First Installed Readmes Should Delay Protocol Language
#learning

## Insight
When an installed workflow README is meant to help people use a deposited tool surface, it works better when plain-language usage guidance comes before protocol, manifest, or contract terminology.

## Reusable Practice
- Start with what the reader is looking at and what they should do next.
- Explain the main working folders in everyday language before introducing machine-readable or contract-oriented details.
- Keep AI bootstrap requirements intact, but move them below the human-first orientation when the file serves both audiences.
