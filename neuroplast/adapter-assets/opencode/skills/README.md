# OpenCode Neuroplast Skills

These files are copy/paste-ready OpenCode skill assets using kebab-case names.

## Included Skills
- `neuroplast-bootstrap`
- `neuroplast-route-short-prompts`
- `neuroplast-execute-act`

## Usage
- Copy these folders into the destination OpenCode skills directory.
- Prefer the bundled OpenCode agents under `neuroplast/adapter-assets/opencode/agents/` so the bootstrap skill is invoked reliably.
- Ensure the bootstrap skill runs before short prompts are interpreted.
